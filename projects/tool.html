Project Overview

In the beginning, TOOL Project was planned only for Mobile App Development service and publishing the application on iOS App Store and Android Play Store. However, over time, I also contributed to branding, website creation, and other design elements, that most of them were done for free as a bonus.

TOOL is a startup based in Prague, Czechia and it is designed to connect local service providers with clients. When people in Prague need a local service (such as painting, construction, landscaping, repairs, etc.), they will no longer have to search on Google, Facebook, or other platforms. This mobile application has a search system based on several specific filters that will let user find a good option.

  • Mobile App Development: Development of one application that includes the utilization of 2 types of users: clients and service providers. Some of the important pages were well explained in the first meeting, but no branding, logo, slogan or any other guidelines were prepared, we did everything together on the way to finishing the application.
  • Website Creation: TOOL website is available at tool-connect.com and it contains multiple pages including information about the platform, features, and download links for the mobile applications.
  • Design: Includes the design of the application, logo, and all the branding and marketing elements that were created.
  • Branding: This included the establishment of branding guidelines and the creation of several marketing elements, such as: logo, slogan, unique-spelling proposition, text for descriptions and features, 5 profile picture options, 5 story/feed/ad posts targeted at service providers, 4 story/feed/ad posts targeted at clients, a YouTube banner, a LinkedIn cover, a Facebook cover, and 2 flyers (front and back) - one for service providers and one for clients.
  • Database Development: Designed and implemented a robust database architecture to support user management, service listings, chat, and other features. The database handles complex relationships between clients and service providers while ensuring data integrity and optimal performance.

Technical Details

TOOL is now built as a fully native mobile application using React Native and Expo, with Supabase as the backend-as-a-service solution. The app uses TypeScript for type safety, Zustand for state management, and TanStack React Query for data fetching and caching. The database is powered by PostgreSQL through Supabase, providing robust authentication, real-time features, and secure data storage. Location services are integrated via Google Maps, and push notifications are handled natively through Expo Notifications. The app supports multiple languages through i18n-js.

The TOOL website was built with Next.js and Tailwind CSS, providing a modern, responsive landing page. The website is now hosted on Netlify and serves as the platform's online presence, providing information about the service, features, and direct links to download the mobile applications from both app stores.

‍The functionalities of the TOOL mobile application are as follows: sign-in options with Google and Apple, automatic push notifications, messaging with attachments, the ability to block and report users, an automated help and support email, availability management for service providers, the ability to leave reviews for service providers, automatic review requests for the App Store and Play Store, the option to add a portfolio project to your service provider profile, and the ability to find service providers in your area using filters. Moreover, an admin dashboard was also needed, including manual notifications, the ability to find and ban users, and analytics for services and user activity.

The marketing strategy was structured into two main segments, each targeting a specific audience. The first segment focused on service providers, including promotional materials designed to help them showcase their expertise, attract clients, and manage their professional presence within the platform. The second segment targeted potential clients, providing engaging content to help them discover, evaluate, and connect with service providers effortlessly. This involved flyers, social media ads, and promotional posts. Key features were highlighted and incorporated into the created posts, including QR codes for app installation, ensuring better transparency and accessibility.

Results

The TOOL application is now live and fully functional, providing a comprehensive platform for connecting local service providers with clients in Prague. Initially available in Prague, Czechia, with plans to expand to other areas. Future updates are planned with focus on improving performance, enhancing flexibility, enabling payment integration, and expanding the website functionality.

The project was completed successfully, delivering a robust native mobile application with a supporting website and comprehensive branding package. I provided a three-month guarantee ensuring that the application is fully functional, during which I will fix any bugs that may arise free of charge.
